# 🌊 Movement Brand Kit

> Your movement identity, distilled by AI.

A brand identity generator built specifically for movement artists, 
athletes, and physical creators. Input your movement philosophy — 
get a complete brand kit instantly.

Built with React, Node.js, and the Anthropic Claude API.

---

## ✨ What It Generates

**Visual Identity**
- Colour palette with hex codes and reasoning for each colour
- Typography pairing — display font and body font
- Aesthetic direction — the visual world of your brand

**Tone of Voice**
- How to write captions that feel authentic
- What language to avoid
- Platform-specific adjustments for Instagram, TikTok, LinkedIn

**Content Strategy**
- 3 content pillars tailored to your movement style
- Posting rhythm with reasoning
- Content formats that suit your practice

**Brand Positioning**
- Your unique angle in one sentence
- How you stand apart from the dominant aesthetic in your space
- A brand manifesto — 2-3 sentences that capture your philosophy
- A one-line Instagram bio, ready to use today

---

## 🎯 Who This Is For

Movement creators who have a practice but haven't found their brand voice yet:
- Calisthenics athletes
- Dancers and contemporary movers
- Acrobats and gymnasts
- Yoga and movement coaches
- Anyone building a personal brand around physical practice

## 🔐 Privacy and Security

Your API key never touches the browser. All Claude API calls are 
made server-side through a Node.js proxy — keeping your credentials 
secure at all times.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- An Anthropic API key ([get one here](https://console.anthropic.com))

### Installation

1. Clone the repository:
```bash
git clone https://github.com/RutvikBhamwari/Movement_Brand_Kit.git
cd Movement_Brand_Kit/brand-kit
```

2. Install dependencies:
```bash
npm install
```

3. Set your API key:
```bash
export ANTHROPIC_API_KEY="your-key-here"
```

4. Start the API server:
```bash
node server.js
```

5. In a new terminal, start the React app:
```bash
npm run dev
```

6. Open your browser at `http://localhost:5173`

## 💡 Design Philosophy

Most brand generators produce generic outputs because they ask 
generic questions. This tool asks about movement philosophy, 
sensation, and aesthetic feeling — not just industry and target market.

The result is a brand kit that feels like it was always yours.

---

## 🎨 Design

Designed in Figma before building — following a design-first approach.

The colour palette (earth tones, warm linen, river blue) was chosen 
to reflect the natural, elemental philosophy of movement creators 
who draw inspiration from water, earth, and open landscapes.
